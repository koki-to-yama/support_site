サポート

CALOGをご利用いただき、ありがとうございます。

🚀 はじめ方ガイド

CALOGを初めて使う方へ、ダウンロードしてすぐ使い始められます！

🎉 すぐに使い始められます！
CALOGは面倒な設定なしで、ダウンロード後すぐにAI解析機能をお使いいただけます。

2つの使い方

CALOGには2つのご利用方法があります：

📱 方法1: すぐに始める（推奨）
共有APIキーで即座にAI解析が使えます。
• 設定不要でダウンロード後すぐに使える
• 写真解析：1日1回
• テキスト解析：1日3回
• 広告視聴で追加利用可能（リワード広告を見ると+1回）
🔑 方法2: 独自APIキーを設定（上級者向け）
ご自身のGoogle Gemini APIキーを設定すると、制限なく使えます。
• 1日の回数制限なし（Googleの無料枠：250回/日）
• 広告表示なし
• 設定方法は下記を参照

【上級者向け】独自APIキーを設定する（任意・5分で完了）

共有APIキーの制限（写真1回/日、テキスト3回/日）を解除したい場合は、ご自身のAPIキーを設定できます。完全無料で取得できます。

💳 クレジットカード登録は不要です！
無料枠のみを使用する場合、クレジットカードや支払い情報の登録は一切不要です。もし登録を求められても、スキップして問題ありません。
  1. Google AI Studioにアクセス
    • こちらのリンクをタップしてください
    • Googleアカウントでログインします（Gmailアドレスがあれば誰でもOK）
    • 初回は利用規約への同意を求められるので、内容を確認して「Agree」または「同意する」をタップ
  2. APIキーを作成（課金なし設定を確認）
    • 「Create API Key」または「APIキーを作成」ボタンをタップ
    • ⚠️ 重要： 「Create API key in new project」（新規プロジェクトでAPIキーを作成）を選択してください
    • 表示されたAPIキー（長い英数字の文字列、例：AIzaSyA...）をコピーします
    • このキーは後で確認できるので、焦らなくてOKです
  3. ✅ 課金されない設定の確認（必須）
    • 確認方法： Google AI Studioの画面で、以下を確認してください
    • ✓ 「Free tier」または「無料枠」と表示されている
    • ✓ 「Billing account」や「支払い方法」の設定画面が表示されていない
    • ✓ クレジットカード情報を入力していない
    ⚠️ もし「Billing account」や「支払い方法」の設定を求められたら：
    • 設定しないでください！ 無料枠のみ使用する場合は不要です
    • スキップまたはキャンセルして、APIキーのみを使用してください
    • 支払い方法を設定すると、将来的に課金される可能性があります
  4. CALOGアプリに設定
    • CALOGアプリを開く
    • 下部メニューから「設定」（⚙️アイコン）をタップ
    • 「APIキー設定」をタップ
    • コピーしたAPIキー（AIzaSyA...で始まる長い文字列）を貼り付けて「保存」
    • 正しく設定できると、ホーム画面で「📷 写真を撮る」が使えるようになります
✅ 課金されないことを再確認：
• クレジットカードを登録していない → ✅ 課金されません
• 無料枠（1日1,500回まで）のみ使用 → ✅ 課金されません
• 無料枠を超えた場合 → ⚠️ 解析が一時的に使えなくなるだけ（自動課金されません）
• 手動入力は常に使用可能 → ✅ API不要で記録できます
🔒 安全性について： APIキーはあなたのiPhone内に安全に保存され、外部には送信されません。

ステップ2: 目標を設定する

あなたの目標カロリーとマクロ栄養素の比率を設定しましょう。

  1. 「設定」画面で「目標設定」を開く
  2. 1日の目標カロリー（例: 2000kcal）を入力
  3. タンパク質・脂質・炭水化物の比率を設定（例: 30%・30%・40%）
  4. 「保存」をタップ

※ わからない場合は、まず2000kcal、30%・30%・40%で始めてみて、後から調整できます。

ステップ3: 食事を記録してみる

準備完了！早速食事を記録してみましょう。

  1. ホーム画面で「📷 写真を撮る」をタップ
  2. 食事をカメラで撮影
  3. AIが自動で栄養素を解析します（数秒かかります）
  4. 結果を確認して「保存」

これだけです！ ホーム画面に今日の進捗が表示されます。

💡 ヒント： 写真は明るい場所で、料理全体がはっきり写るように撮ると、より正確に解析されます。

🆓 無料で使える範囲は？

📱 共有APIキー（設定不要）の場合

  • 写真解析：1日1回
  • テキスト解析：1日3回
  • 毎日0時にリセット
  • 広告を視聴すると追加で1回利用可能

※ 共有APIキーは多くのユーザーで共有しているため、公平に利用いただけるよう制限を設けています。

🔑 独自APIキーを設定した場合

Google Gemini APIの無料枠（2025年12月時点）：

  • 1日最大250回のリクエスト（Flashモデル）
  • 1分間に10回まで解析可能
  • 広告表示なし

※ 1日3食×30日=90回程度の使用が一般的なので、無料枠（250回/日）で十分カバーできます。

⚠️ 重要： 無料枠やレート制限はGoogleの仕様変更により今後変わる可能性があります。最新情報はGoogle AI 公式ドキュメントでご確認ください。

よくある質問（FAQ）

Q1. アプリが起動しない / クラッシュする

以下の手順をお試しください。

  1. アプリを完全に終了し、再起動してください
  2. デバイスを再起動してください
  3. アプリを最新バージョンに更新してください
  4. それでも解決しない場合は、お問い合わせください

Q2. 写真から栄養素が正しく解析されない

より正確な解析結果を得るために、以下をお試しください。

  • 食事全体が明るく、はっきりと写るように撮影してください
  • 複数の料理がある場合は、個別に撮影することをお勧めします
  • 解析結果は参考値です。必要に応じて手動で修正してください

Q3. 料金はかかりますか？広告はありますか？

CALOGは完全無料でご利用いただけます。

  • CALOGアプリ自体は完全無料です
  • アプリ内課金はありません
  • アカウント登録も不要です

📺 広告について：

CALOGでは、アプリを無料で提供し続けるために広告を表示しています。

  • リワード広告： 視聴するとAI解析の追加利用回数がもらえます（任意）
  • インタースティシャル広告： 画面切り替え時に表示されることがあります

※ 独自のAPIキーを設定すると、広告は表示されません。

📌 AI解析の利用回数：

  • 共有APIキー（初期設定）： 写真1回/日、テキスト3回/日
  • 独自APIキー設定後： 250回/日（Googleの無料枠）
  • ✅ 手動入力は常に無制限で使用可能
✅ 安心してお使いください：
• CALOGからの課金請求は一切ありません
• 独自APIキー使用時も、クレジットカードを登録しなければGoogleへの課金もありません
• 手動入力は常に使用可能です

Q4. APIキーの設定方法は？

詳しくは上記の「はじめ方ガイド」をご覧ください。簡単に言うと：

  1. Google AI StudioでAPIキーを無料取得
  2. アプリの「設定」→「APIキー設定」で貼り付け
  3. 保存（デバイス内に安全に保存されます）

Q5. 目標設定を変更したい

「設定」画面から、いつでも目標カロリーとマクロ栄養素の目標を変更できます。

Q6. 過去の記録を修正・削除したい

「食事一覧」画面から過去の記録を選択し、編集または削除することができます。

Q7. 体重記録はどう使いますか？

「体重」タブから体重を記録できます。体重の推移をグラフで確認し、目標達成に役立てることができます。

Q8. 手動入力の方法は？

写真を撮れない場合や、より詳細に記録したい場合は手動入力が可能です。2つの方法があります：

方法1: テキストからAI解析（おすすめ）

  1. ホーム画面で「✏️ 手動入力」をタップ
  2. 「✨ テキストから解析」をタップ
  3. 食品名を入力（例：「鶏むね肉200g」「サラダチキン1個」）
  4. AIが自動で栄養素を推定します
  5. 結果を確認して「保存」

💡 ヒント：量や調理方法も含めて入力すると、より正確に解析されます。

方法2: 完全手動入力

  1. ホーム画面で「✏️ 手動入力」をタップ
  2. 食品名、カロリー、タンパク質、脂質、炭水化物を直接入力
  3. 「保存」をタップ

※ パッケージに記載されている栄養成分表示を参考にすると正確です。

Q9. 解析結果が間違っている場合は？

AI解析は推定値のため、100%正確ではありません。以下の方法で対応できます：

  • 解析結果を編集して修正する
  • 削除して手動入力で記録し直す
  • 複数の料理がある場合は、個別に撮影すると精度が上がります

Q10. 筋トレや運動の記録はどう使いますか？

ホーム画面の「💪 筋トレ」から、ウェイトトレーニングや有酸素運動を記録できます。

  • 筋トレ： セッションを開始し、種目・重量・回数・セット数を記録
  • 有酸素運動： ランニングや水泳などの運動と時間を記録
  • 消費カロリー： MET値に基づいて自動計算されます

※ 消費カロリーは推定値です。運動による消費カロリーは個人差があります。

Q11. おすすめアイテムとは？

ダイエットや筋トレに役立つ商品（プロテイン、サプリメント、トレーニング用品など）を紹介しています。

  • ホーム画面の「🛒 おすすめ」からアクセス
  • 商品をタップすると販売サイト（外部サイト）に移動します
  • アフィリエイトリンク（PR）を含みます
  • 利用は任意で、アプリの機能に影響しません

📱 機能説明と使い方

📷 カメラ解析（AI自動解析）

カメラで食事を撮影すると、AI（Google Gemini）が自動的に食品を認識し、栄養素を推定します。

使い方：

  1. ホーム画面の「📷 写真を撮る」をタップ
  2. 食事をカメラで撮影（明るい場所で、料理全体がはっきり写るように）
  3. AIが自動で解析（数秒～10秒程度）
  4. 結果を確認して「保存」

💡 ヒント：複数の料理がある場合は個別に撮影すると、より正確に解析されます。

✨ テキスト解析（AI解析）

写真を撮らずに、食品名をテキストで入力するだけでAIが栄養素を推定します。写真が撮れない状況や、素早く記録したい時に便利です。

使い方：

  1. ホーム画面の「✏️ 手動入力」をタップ
  2. 「✨ テキストから解析」をタップ
  3. 食品名と量を入力（例：「鶏むね肉200g」「白米茶碗1杯」「サラダチキン1個」）
  4. AIが自動で栄養素を推定（数秒程度）
  5. 結果を確認して「保存」

💡 ヒント：量や調理方法（焼き、蒸し、揚げなど）も含めて入力すると、より正確に解析されます。

✏️ 完全手動入力

栄養成分表示をもとに正確に記録したい場合や、AI解析を使わずに自分で入力したい場合に使えます。

使い方：

  1. ホーム画面の「✏️ 手動入力」をタップ
  2. 食品名と栄養素（カロリー、P/F/C）を直接入力
  3. 「保存」をタップ

💡 ヒント：コンビニ食品やパッケージ食品は、裏面の栄養成分表示を見ながら入力すると正確です。

📊 日次進捗（ホーム画面）

ホーム画面で、その日の目標に対する進捗をリアルタイムで確認できます。

  • カロリー： 目標に対する摂取カロリーの割合
  • タンパク質（P）： 目標に対するタンパク質の達成率
  • 脂質（F）： 目標に対する脂質の達成率
  • 炭水化物（C）： 目標に対する炭水化物の達成率

各栄養素のバーが色分けされており、目標に近づいているか一目でわかります。

📝 食事履歴（食事一覧タブ）

過去の食事記録を日付ごとに確認できます。

できること：

  • 過去の記録を日付で確認
  • 各記録をタップして詳細表示
  • 記録の編集（栄養素を修正）
  • 記録の削除

⚖️ 体重記録（体重タブ）

定期的に体重を記録することで、栄養管理と合わせて健康管理に役立てることができます。

使い方：

  1. 下部メニューから「体重」タブをタップ
  2. 「体重を記録」をタップ
  3. 今日の体重を入力（例：65.5kg）
  4. 「保存」をタップ

記録した体重はグラフで表示され、推移が一目でわかります。

⚙️ 設定画面

目標設定やAPIキーの管理ができます。

  • APIキー設定： Gemini APIキーの登録・変更
  • 目標設定： 目標カロリーとPFC比率の設定
  • アプリ情報： バージョン確認

💪 筋トレ記録

ウェイトトレーニングや自重トレーニングを記録し、消費カロリーを自動計算します。200種類以上のエクササイズがプリセットされています。

使い方：

  1. ホーム画面の「💪 筋トレ」をタップ
  2. 「セッションを開始」をタップしてトレーニング開始
  3. 種目を選択（部位別に分類：胸・背中・肩・腕・脚・体幹）
  4. 重量・回数・セット数を入力
  5. トレーニング終了後「セッション終了」をタップ

プリセット種目の例：

  • 胸： ベンチプレス、ダンベルフライ、プッシュアップなど
  • 背中： デッドリフト、ラットプルダウン、懸垂など
  • 肩： ショルダープレス、サイドレイズなど
  • 腕： バイセップカール、トライセップエクステンションなど
  • 脚： スクワット、レッグプレス、ランジなど
  • 体幹： プランク、クランチ、レッグレイズなど

💡 消費カロリーはMET値（運動強度の指標）をもとに自動計算されます。詳細はエビデンス・参考情報をご覧ください。

🏃 有酸素運動記録

ランニング、ウォーキング、サイクリングなどの有酸素運動を記録できます。

使い方：

  1. ホーム画面の「💪 筋トレ」→「有酸素運動」タブをタップ
  2. 運動の種類を選択（ランニング、ウォーキング、サイクリング、水泳など）
  3. 運動時間を入力
  4. 「記録」をタップ

対応する有酸素運動：

  • ランニング（各種ペース）
  • ウォーキング
  • サイクリング
  • 水泳
  • 縄跳び
  • エアロバイク
  • その他のカーディオマシン

💡 消費カロリーは運動種類と時間から自動計算されます。

🛒 おすすめアイテム

ダイエットや筋トレに役立つおすすめ商品を紹介しています。

使い方：

  1. ホーム画面の「🛒 おすすめ」をタップ
  2. 興味のある商品をタップして詳細を確認
  3. 商品ページで購入（外部サイトに移動します）
📌 アフィリエイトについて：
おすすめアイテム画面には、アフィリエイトリンク（PR）が含まれています。リンク経由で商品を購入された場合、当社が紹介料を受け取ることがあります。商品の品質や価格については、各販売サイトでご確認ください。なお、アフィリエイトリンクの利用は任意であり、アプリの基本機能には影響しません。

動作環境

  • 対応機種: iOS 13.0以上のiPhone
  • 画像解析: Google AI（Gemini）を使用
  • 広告配信: Google AdMobを使用
  • データ保存: すべてあなたのiPhone内に保存
  • セキュリティ: APIキーはiPhoneの安全な保存領域で保護

プライバシーとセキュリティ

  • すべてのデータ（食事記録、体重、筋トレ記録、目標設定）はあなたのiPhone内にのみ保存されます
  • 写真は解析時のみGoogle AIに送信され、当社では保存しません
  • APIキーはiPhoneの安全な保存領域に暗号化して保存されます
  • 広告配信のため、広告識別子（IDFA）がGoogle AdMobに送信される場合があります
  • 個人情報を第三者に提供することはありません

詳細はプライバシーポリシーをご覧ください。

ご利用時の注意点

  • 複数の料理が写っている写真では、解析の精度が下がる場合があります（個別に撮影することをおすすめします）
  • 短時間に連続して解析すると、一時的に利用できなくなることがあります（無料枠の制限のため）
💡 ヒント： AIモデルによって無料枠が異なります（2025年1月時点）：
• Flash版：1分間10回、1日250回
• Pro版：1分間2回、1日50回
高度な設定からモデルを選択できます。
